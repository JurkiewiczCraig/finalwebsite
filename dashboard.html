Dashboard for EPA National Walkability Index

Go to Dashboard
Why Dashboard?
Dashboards are composed of configurable elements, such as maps, lists, charts, gauges, indicators, and tables. Most elements are data driven and thus makes it ideal to showcase the walkability index factors in a way that the intended audience can understand.
Positives Negatives
Performance when large datasets are used Limited Customization which further requires arcade coding
Customization of layouts and themes Requires time to explore complex configurations
Accessibility from any device Not available without internet connection
Integration with other ArcGIS tools and data sources Needs subscription/lisence
Ability to integrate real-time data
What was intresting?
The most intresting part to work on this dashboard was to colorcode the text in the list with the colors assigned for the respective walkability classification using arcade coding.
The dashboard was designed for mobile devices too and had arranged the necessary parameters.
