Leaflet

Leaflet is an open source javascript library for mobile-friendly interactive maps. For a possible solution to our problem statement , our team attempted to create a web map (without the use of any API) to show off walkability index's within the united states. This map would include a geolocator as well so users could walk around their own neighborhood and see how well their walkability indexes compare to other areas. Below we have a list of what worked well with this potential solution and what issues came up. Along with potential ideas of what other things could be implemented in future designs

Check out our Leaflet Map

What Worked Well

  • Using Open Source data meant I didn't need to fuss to API keys and could manipulate the data as I saw fit
  • Map was usable on both desktop and mobile for users
  • Relatively easy to pick up for anyone who has basic skills in coding
  • Can easily be worked on collaboratively as a group in Github and VSCode

What Went wrong

  • Limitations on file size when using GeoJSon data, required a lot of time cutting down on data to ensure it would fit into GitHub as well ensuring the file itself could be modified for use
  • Required more time than anticipated to relearn HTML, CSS and Java when going over the online tutorials
  • Limited interactivity and capabilities when compared to Esri Web Maps, would have been able to create similar level of interactivity using proprietary sources with half as much time being spent on creating them
  • With the project timeline, was only able to include one layer showcasing the walkability idnex, and wasn't able to include any indepth levels of interactivity (pop ups, search bars etc)

Future Possibilies

    >
  • When working with larger datasets(such as the one we focused this project on) would be better to not use GeoJSon data, and focus more on using Esri's API keys instead
    • Future versions would include more functionality within the map, such as interactive pop ups and a search bar
  • As cities grow and adapt, their walkability scores change, so this map could easily be updated as time goes on and more and more data becomes available
  • Future iterations of this potential solution would include several layers showcasing the four main factors associated with calculating the walkability index and have users comapre and contrast between them
