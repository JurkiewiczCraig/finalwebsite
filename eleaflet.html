Leaflet

Leaflet is an open source javascript library for mobile-friendly interactive maps. For a possible solution to our problem statement, our team attempted to create a web map (without the use of any API) to show off walkability index's within the united states. This map would include a geolocator as well so civillians could walk around their own neighborhood and see how well their walkability indexes compare to other areas. Below we have a list compiled of what worked well for this potential solution and what could be improved upon

Positives

  • Open source
  • Mobile and Desktop Use
  • Anyone has the ability to edit/manipulate and create their own version
  • Can earily update map as new data becomes avialable

Negatives

  • Limited file size with GeoJSon data
  • Limited interactivity and capabilities when compared to Esri Web Maps
  • Requires prior knowledge of coding with HTML, CSS and Java
  • Data is open to the public, data is not as secure when compared to other software solutions
  • If more time was alotted for the project, some other factors could have been involved, including more data being presented along with web layers
  • Hard to get full grasp of capabilities with geolocator, unless you're in the area where the data is, the map isn't as functional as it should be
