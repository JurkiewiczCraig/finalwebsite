Leaflet

Leaflet is an open source javascript library for mobile-friendly interactive maps. For a possible solution to our problem statement , our team attempted to create a web map (without the use of any API) to show off walkability index's within the united states. This map would include a geolocator as well so users could walk around their own neighborhood and see how well their walkability indexes compare to other areas. Below we have a list of what worked well with this potential solution and what issues came up. Along with potential ideas of what other things could be implemented in future designs

Check out our Leaflet Map

Positives

  • Open source
  • Mobile and Desktop Use
  • Anyone has the ability to edit/manipulate the data if they wanted to
  • Relatively easy to pick up for anyone who has basic skills in coding
  • Can easily be worked on collaboratively

Negatives

  • Limited file size with GeoJSon data
  • Limited interactivity and capabilities when compared to Esri Web Maps
  • Requires prior knowledge of coding with HTML, CSS and Java
  • Data is open to the public, so would need to assess data input priot to bringing in to ensure security
  • If more time was alotted for the project, some other factors could have been involved, including more data being presented along with web layers
  • With project timeline, was only able to include one layer showcasing the walkability idnex, future attempts with more time would allow the team to add in more layers, search functions and other aspects to the map to make it more interactive
